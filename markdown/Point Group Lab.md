# Point Group Lab 工具介绍

这个页面是一个晶体学点群交互可视化工具，核心目标是把“点群操作”从抽象的群论符号，变成可以直接拖拽、缩放、切换参数观看的三维图像。它适合用来快速理解旋转轴、镜面、反轴、反演中心，以及它们作用后生成的轨道点和凸包结构。

## 1. 这个工具能做什么

Point Group Lab 主要用来展示晶体学点群的几何结果。它会根据用户输入的生成元，自动构造整个对称群的闭包，并把初始点在群作用下生成的所有等价点画出来。

你可以把它理解为三件事的组合：

1. 输入对称操作。
2. 计算点群闭包和轨道点。
3. 用可视化方式展示对称元素、凸包、坐标轴和轨迹线。

## 2. 页面里的主要功能

### 2.1 初始点与生成元

页面左侧可以设置一个初始三维点，以及多个常见生成元：

- 三个主坐标轴上的旋转或反轴。
- 沿 [111] 方向的旋转或反轴。
- 三个正交镜面。
- 中心反演。

工具会据此生成对称群，并计算点的轨道。

### 2.2 预设组合

为了方便快速演示，页面内置了一些常见点群预设，例如：

- `C1`
- `Ci`
- `Cs`
- `C2`
- `C3`
- `C4`
- `C6`
- `C2v`
- `D2h`
- `m3-bar (T_h)`

选择预设后，工具会自动填入对应的生成元，便于一键查看典型点群结构。

### 2.3 可视化开关

你可以控制页面里显示的内容：

- 显示坐标轴。
- 显示对称元素。
- 显示凸包表面。
- 调节凸包透明度。
- 切换标准图示模式。
- 显示原点连线。
- 开启自动旋转视角。

这些选项适合在“教学展示”和“细节分析”两种模式之间切换。

### 2.4 视角交互

主可视化区域支持常见三维交互：

- 拖拽旋转视角。
- 滚轮缩放。
- 双击重置视角。
- 全屏查看可视化。

全屏模式尤其适合在对称元素很多、点群结构比较密集的时候观察细节。

## 3. 这个工具展示了什么数学内容

虽然它是一个可视化工具，但底层仍然是标准的点群数学模型。核心思想是：把旋转、镜像、反演、反轴都写成三维矩阵变换，然后对初始点做群作用，得到整个轨道集合。

### 3.1 群作用与轨道

如果初始点是 \(x\)，点群中的操作是 \(g\)，那么生成的点就是：

\[
g(x)
\]

对所有操作都应用一遍之后，就得到轨道集合，也就是图上看到的一簇对称等价点。

### 3.2 对称元素的遍历

工具不仅画点，还会根据闭包后的群操作自动推断对称元素，包括：

- 旋转轴。
- 反轴。
- 镜面。
- 反演中心。

这使得页面不只是“画出结果”，还会“解释结果是怎么来的”。

### 3.3 凸包与遮挡

当轨道点足够多时，工具会计算三维凸包，并把它作为一个半透明表面显示出来。这样可以更直观看到点群生成的空间包络，也能帮助理解哪些点处于前景、哪些点被遮挡。

## 4. 适合怎么用

这个工具比较适合以下场景：

- 讲解晶体学点群的基础概念。
- 对照标准符号理解旋转轴、镜面和反演。
- 检查某个生成元组合会得到怎样的轨道结构。
- 在网页里快速展示一个点群的空间对称性。

如果你只是想快速演示，可以直接选预设。如果你想研究更细的结构，可以手动修改轴阶数、镜面和初始点位置，再观察变化。

## 5. 设计目标

这个页面的设计目标不是“只给出一个结果图”，而是尽量接近一个可讲解、可演示、可比较的教学工具：

- 保持晶体学符号尽量标准。
- 让对称元素看起来像常见教材和 MATLAB 脚本中的表现方式。
- 让界面在桌面端和移动端都能正常查看。
- 让图像在全屏和普通卡片模式下都保持清晰。

## 6. 一句话总结

Point Group Lab 是一个把晶体学点群“算出来、画出来、讲出来”的交互工具：输入生成元，自动构造点群闭包，再用三维可视化把轨道点、对称元素和凸包统一展示出来。
