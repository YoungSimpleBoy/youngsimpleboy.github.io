GitHub Pages 博客搭建指南

本网页由minmax生成，感谢伟大的C语言开源。

本文档详细介绍如何使用 GitHub Pages 搭建个人博客，完全使用 HTML 文件，无需 Jekyll 或其他静态网站生成器。

博客地址： https://youngsimpleboy.github.io

一、创建 GitHub 仓库

步骤 1：创建仓库

  1. 登录 GitHub
  2. 点击右上角 "+" → "New repository"
  3. 仓库名称填写：你的用户名.github.io
  4. 选择 Public
  5. 点击 "Create repository"
注意：仓库名称必须是 用户名.github.io 格式，例如 youngsimpleboy.github.io

二、上传博客文件

文件结构

你的仓库/
├── index.html          # 首页
├── style.css          # 样式文件
├── script.js          # 脚本文件
├── avatar.png         # 头像图片（可选）
└── posts/             # 文章文件夹
    └── hello-world.html     # 文章页面

上传方式

  1. 进入仓库页面
  2. 点击 "Add file" → "Upload files"
  3. 拖拽或选择要上传的文件
  4. 点击 "Commit changes" 提交
重要：不要上传 README.md 文件，否则会覆盖 index.html 的显示！

三、启用 GitHub Pages

  1. 进入仓库 → Settings → Pages
  2. 在 "Build and deployment" 部分，找到 "Source"
  3. 选择 "Deploy from a branch"
  4. Branch 选择 "main"，目录选择 "/(root)"
  5. 点击 "Save"

等待 1-2 分钟，访问 https://你的用户名.github.io 即可看到博客。

四、文件说明

文件 说明
index.html 博客首页，显示文章列表
style.css 样式文件，控制页面外观
script.js 脚本文件，实现交互功能（深色模式等）
avatar.png 头像图片，显示在首页简介处
posts/ 存放文章 HTML 文件的文件夹

五、写新文章

1. 创建文章文件

在 posts/ 文件夹中创建新的 HTML 文件，例如 posts/new-article.html：

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>文章标题 - 博客名</title>
    <link rel="stylesheet" href="../style.css">
</head>
<body>
    <!-- 导航栏 -->
    <header>...</header>
    
    <!-- 文章内容 -->
    <main class="main-content article-content">
        <article>
            <h1>文章标题</h1>
            <div class="article-body">
                <p>正文内容...</p>
            </div>
        </article>
    </main>
    
    <!-- 页脚 -->
    <footer>...</footer>
    
    <script src="../script.js"></script>
</body>
</html>

2. 在首页添加链接

在 index.html 的文章列表中添加：

<article class="post-item">
    <div class="post-meta">
        <time>2026-03-15</time>
        <span class="post-category">分类名</span>
    </div>
    <h3 class="post-title">
        <a href="posts/new-article.html">文章标题</a>
    </h3>
    <p class="post-excerpt">文章摘要...</p>
</article>

六、个性化修改

修改个人信息

在 index.html 中修改：

<!-- 修改名字 -->
<h1 class="profile-name">你的名字</h1>

<!-- 修改简介 -->
<p class="profile-bio">你的简介</p>

<!-- 修改描述 -->
<p class="profile-desc">在这里记录学习笔记。</p>

修改社交链接

在 index.html 的页脚部分：

<div class="footer-links">
    <a href="https://github.com/你的用户名">GitHub</a>
    <a href="mailto:your@email.com">邮箱</a>
    <a href="https://bilibili.com/你的ID">Bilibili</a>
</div>

修改头像

将你的头像图片命名为 avatar.png，或修改 index.html 中的图片路径：

<img src="你的头像文件.jpg" alt="头像" class="profile-avatar">

七、常见问题

Q1: 访问网站显示的是 README 而不是 index.html？

删除仓库中的 README.md 文件即可。

Q2: 网站显示旧内容？

按 Ctrl + Shift + R（Windows）或 Cmd + Shift + R（Mac）强制刷新清除缓存。

Q3: 图片显示不了？

检查图片路径是否正确，确保图片已上传到仓库根目录。

Q4: 如何使用 Markdown 写文章？

可以使用在线工具（如 Markdown to HTML）将 Markdown 转换为 HTML，然后保存到 posts/ 文件夹。

八、主题切换

博客支持深色/浅色模式切换。点击导航栏中的太阳/月亮图标即可切换主题。主题设置会保存在浏览器本地，下次访问时自动记忆。
