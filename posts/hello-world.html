GitHub Pages 博客搭建指南

博客 建站

网页由minmax生成，感谢伟大的C语言开源。

本文档详细介绍如何使用 GitHub Pages 搭建个人博客，完全使用 HTML 文件，无需 Jekyll 或其他静态网站生成器。

一、创建 GitHub 仓库

登录 GitHub，点击右上角 "+" → "New repository"，仓库名称填写：你的用户名.github.io，选择 Public，点击 "Create repository"。

注意：仓库名称必须是 用户名.github.io 格式，例如 youngsimpleboy.github.io

二、上传博客文件

文件结构如下：

你的仓库/
├── index.html          # 首页
├── style.css          # 样式文件
├── script.js          # 脚本文件
├── avatar.png         # 头像图片
└── posts/             # 文章文件夹

三、启用 GitHub Pages

  1. 进入仓库 → Settings → Pages
  2. 在 "Build and deployment" 部分，找到 "Source"
  3. 选择 "Deploy from a branch"
  4. Branch 选择 "main"，目录选择 "/(root)"
  5. 点击 "Save"

四、写新文章

在 posts/ 文件夹中创建新的 HTML 文件，引用上级目录的样式表：

<link rel="stylesheet" href="../style.css">

五、主题切换

博客支持深色/浅色模式切换。点击导航栏中的太阳/月亮图标即可切换主题。主题设置会保存在浏览器本地，下次访问时自动记忆。
