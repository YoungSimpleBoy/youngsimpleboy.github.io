三维空间旋转变换与平面镜像变换 理论证明

线性代数

第一部分：三维绕定点定向量旋转变换证明

一、符号定义

设三维欧氏空间 R 3 \mathbb{R}^3 R3 中，所有向量、点均为列向量，定义如下：

  • 原始点集矩阵： A ∈ R 3 × N A \in \mathbb{R}^{3 \times N} A∈R3×N，每一列对应一个空间点 a a a

  • 旋转中心点： p ∈ R 3 p \in \mathbb{R}^3 p∈R3

  • 旋转轴方向向量： v ∈ R 3 v \in \mathbb{R}^3 v∈R3（非零向量），单位化后为 v ^ \hat{v} v^

  • 旋转角度： θ \theta θ（弧度制，逆时针为正，遵循右手定则）

  • 旋转后点集矩阵： B ∈ R 3 × N B \in \mathbb{R}^{3 \times N} B∈R3×N，每一列对应旋转后点 b b b

  • 单位矩阵： I ∈ R 3 × 3 I \in \mathbb{R}^{3 \times 3} I∈R3×3；向量模长： ∥ ⋅ ∥ \| \cdot \| ∥⋅∥；向量叉乘： × \times ×

二、核心变换思路

绕空间定点 p p p 的旋转变换，需通过平移归一化→绕原点旋转→逆平移还原，将非标旋转转化为标准原点旋转。

三、严格推导过程

1. 平移归一化（旋转中心移至原点）

将所有点沿 − p -p −p 平移，使旋转中心 p p p 与坐标原点重合，平移后点为：

a ′ = a − p a' = a - p a′=a−p

此时绕 p p p 旋转等价于绕原点对 a ′ a' a′ 旋转。

2. 旋转轴单位化

Rodriguez公式要求旋转轴为单位向量，因此对方向向量归一化：

v ^ = v ∥ v ∥ \hat{v} = \frac{v}{\| v \|} v^=∥v∥v​

3. Rodriguez旋转公式推导

将向量 a ′ a' a′ 分解为平行于旋转轴的分量 a ∥ ′ a'_\parallel a∥′​ 和垂直于旋转轴的分量 a ⊥ ′ a'_\perp a⊥′​：

a ′ = a ∥ ′ + a ⊥ ′ a' = a'_\parallel + a'_\perp a′=a∥′​+a⊥′​

  • 平行分量（旋转后不变）： a ∥ ′ = ( v ^ T a ′ ) v ^ a'_\parallel = (\hat{v}^\mathrm{T} a') \hat{v} a∥′​=(v^Ta′)v^

  • 垂直分量（参与平面旋转）： a ⊥ ′ = a ′ − a ∥ ′ a'_\perp = a' - a'_\parallel a⊥′​=a′−a∥′​

垂直分量在垂直于 v ^ \hat{v} v^ 的平面内旋转 θ \theta θ 后，结果为：

a ⊥ ′ ( θ ) = a ⊥ ′ cos ⁡ θ + ( v ^ × a ⊥ ′ ) sin ⁡ θ a'_\perp(\theta) = a'_\perp \cos\theta + (\hat{v} \times a'_\perp) \sin\theta a⊥′​(θ)=a⊥′​cosθ+(v^×a⊥′​)sinθ

合并分量并化简，得到绕原点的旋转向量公式：

a ′ ′ = a ′ cos ⁡ θ + ( 1 − cos ⁡ θ ) ( v ^ T a ′ ) v ^ + sin ⁡ θ ( v ^ × a ′ ) a'' = a' \cos\theta + (1-\cos\theta)(\hat{v}^\mathrm{T} a')\hat{v} + \sin\theta (\hat{v} \times a') a′′=a′cosθ+(1−cosθ)(v^Ta′)v^+sinθ(v^×a′)

4. 旋转矩阵形式

定义 v ^ \hat{v} v^ 的反对称叉乘矩阵：

[ v ^ ] × = ( 0 − v ^ z v ^ y v ^ z 0 − v ^ x − v ^ y v ^ x 0 ) [\hat{v}]_\times = \begin{pmatrix} 0 & -\hat{v}_z & \hat{v}_y \\ \hat{v}_z & 0 & -\hat{v}_x \\ -\hat{v}_y & \hat{v}_x & 0 \end{pmatrix} [v^]×​=​0v^z​−v^y​​−v^z​0v^x​​v^y​−v^x​0​​

Rodriguez旋转矩阵为：

R = cos ⁡ θ ⋅ I + ( 1 − cos ⁡ θ ) ⋅ v ^ v ^ T + sin ⁡ θ ⋅ [ v ^ ] × R = \cos\theta \cdot I + (1-\cos\theta) \cdot \hat{v}\hat{v}^\mathrm{T} + \sin\theta \cdot [\hat{v}]_\times R=cosθ⋅I+(1−cosθ)⋅v^v^T+sinθ⋅[v^]×​

绕原点旋转结果： a ′ ′ = R a ′ a'' = R a' a′′=Ra′

5. 逆平移还原

将旋转后的点平移回原坐标系，最终旋转点：

b = a ′ ′ + p = R ( a − p ) + p b = a'' + p = R(a - p) + p b=a′′+p=R(a−p)+p

6. 点集矩阵批量运算

B = R ( A − p 1 T ) + p 1 T B = R(A - p\mathbf{1}^\mathrm{T}) + p\mathbf{1}^\mathrm{T} B=R(A−p1T)+p1T

其中 1 T \mathbf{1}^\mathrm{T} 1T 为全1行向量，实现广播平移。

第二部分：三维点关于平面的镜像变换证明

一、符号定义

设三维欧氏空间 R 3 \mathbb{R}^3 R3 中，所有向量、点均为列向量，定义如下：

  • 镜面平面由三个不共线点确定： p 1 , p 2 , p 3 ∈ R 3 p_1, p_2, p_3 \in \mathbb{R}^3 p1​,p2​,p3​∈R3

  • 三点坐标矩阵： A = [ p 1 p 2 p 3 ] ∈ R 3 × 3 A = \begin{bmatrix} p_1 & p_2 & p_3 \end{bmatrix} \in \mathbb{R}^{3 \times 3} A=[p1​​p2​​p3​​]∈R3×3

  • 待镜像点集矩阵： B ∈ R 3 × n B \in \mathbb{R}^{3 \times n} B∈R3×n，每一列对应待镜像点 b b b

  • 镜像后点集矩阵： C ∈ R 3 × n C \in \mathbb{R}^{3 \times n} C∈R3×n，每一列对应镜像点 c c c

  • 平面单位法向量： n ^ ∈ R 3 \hat{n} \in \mathbb{R}^3 n^∈R3，满足 ∥ n ^ ∥ = 1 \| \hat{n} \| = 1 ∥n^∥=1

  • 平面方程参数： d d d；全1行向量： 1 1 × n \mathbf{1}_{1 \times n} 11×n​

二、平面方程推导

1. 构造平面内向量

取三点构造两个不共线的平面切向量：

v 1 = p 2 − p 1 , v 2 = p 3 − p 1 v_1 = p_2 - p_1, \quad v_2 = p_3 - p_1 v1​=p2​−p1​,v2​=p3​−p1​

三点不共线充要条件： v 1 × v 2 ≠ 0 v_1 \times v_2 \neq 0 v1​×v2​=0

2. 求解平面法向量

法向量由叉乘得到，再归一化为单位向量：

n = v 1 × v 2 , n ^ = n ∥ n ∥ n = v_1 \times v_2, \quad \hat{n} = \frac{n}{\| n \|} n=v1​×v2​,n^=∥n∥n​

3. 平面标准方程

平面内任意点 x x x 满足法向量与平面内向量垂直，即：

n ^ T ( x − p 1 ) = 0 \hat{n}^\mathrm{T} (x - p_1) = 0 n^T(x−p1​)=0

展开并令 d = − n ^ T p 1 d = -\hat{n}^\mathrm{T} p_1 d=−n^Tp1​，得到平面方程：

n ^ T x + d = 0 \hat{n}^\mathrm{T} x + d = 0 n^Tx+d=0

三、单点镜像公式严格证明

点 b b b 关于平面的镜像点 c c c 满足两个几何约束：

  1. 中点 m = b + c 2 m = \dfrac{b + c}{2} m=2b+c​ 落在镜面上；

  2. 连线向量 c − b c - b c−b 与平面法向量 n ^ \hat{n} n^ 平行。

Step1：代入中点约束

中点满足平面方程：

n ^ T ( b + c 2 ) + d = 0 \hat{n}^\mathrm{T} \left( \frac{b + c}{2} \right) + d = 0 n^T(2b+c​)+d=0

整理得：

n ^ T ( b + c ) + 2 d = 0 (1) \hat{n}^\mathrm{T} (b + c) + 2d = 0 \tag{1} n^T(b+c)+2d=0(1)

Step2：代入平行约束

存在标量 λ \lambda λ，使得：

c − b = λ n ^ (2) c - b = \lambda \hat{n} \tag{2} c−b=λn^(2)

Step3：联立求解标量 λ \lambda λ

将式(2)代入式(1)，结合 n ^ T n ^ = 1 \hat{n}^\mathrm{T} \hat{n} = 1 n^Tn^=1 化简：

2 n ^ T b + λ + 2 d = 0 2\hat{n}^\mathrm{T} b + \lambda + 2d = 0 2n^Tb+λ+2d=0

解得：

λ = − 2 ( n ^ T b + d ) \lambda = -2\left( \hat{n}^\mathrm{T} b + d \right) λ=−2(n^Tb+d)

Step4：镜像公式最终形式

将 λ \lambda λ 代回式(2)，得到单点镜像公式：

c = b − 2 ( n ^ T b + d ) n ^ c = b - 2\left( \hat{n}^\mathrm{T} b + d \right) \hat{n} c=b−2(n^Tb+d)n^

四、点集矩阵批量运算

对批量点集 B B B 做矩阵化运算，避免逐点计算：

  1. 计算所有点到平面的有向距离行向量： s = n ^ T B + d ⋅ 1 1 × n s = \hat{n}^\mathrm{T} B + d \cdot \mathbf{1}_{1 \times n} s=n^TB+d⋅11×n​

  2. 批量镜像变换矩阵公式： C = B − 2 n ^ s C = B - 2\hat{n} s C=B−2n^s

五、几何意义说明

n ^ T b + d \hat{n}^\mathrm{T} b + d n^Tb+d 表示点 b b b 到镜面的距离，镜像变换本质是：点沿法线方向投影至平面，再沿法线反向等距延伸，最终得到对称点。
